# Sudoku (Installable App)

This folder is ready to upload to a GitHub repository and turn on GitHub Pages.

## GitHub Pages setup
1. Create a new GitHub repo (example name: `sudoku`)
2. Upload **all files in this folder** (index.html, style.css, script.js, manifest.json, service-worker.js, icons)
3. Go to **Settings → Pages**
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Save — GitHub will give you a public URL.

## Install on Android
Open the URL in Chrome → menu (⋮) → **Install app** (or Add to Home screen).

## Install on iOS
Open the URL in Safari → Share button → **Add to Home Screen**.
